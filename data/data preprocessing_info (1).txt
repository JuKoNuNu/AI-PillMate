# 진행한 데이터 전처리
### 1. **필요 없는 행/열 제거 및 정리**

| 작업 내용 |
| `제품명`이 없는 행 제거 |
| `이미지_URL`, `이미지_설명`이 없는 행 제거 |
| `보험코드`, `분할선`, `사용상의주의사항(전문가)` 등 분석에 필요 없는 열 제거 |
| `제품명` 열 삭제 후 `이미지_설명`을 제품명으로 변경 |
| '분할선' 열 삭제: 분석에 불필요한 정보 제거 |

### 2. **`크기` 열 분리**

| 작업 내용 | 설명 |
| `크기` → `장축`, `단축`, `두께`로 분리 | 모양 기반 식별에 유용 (예: 타원형, 두께 기준 검색 등) |

### 3. **주의사항 통합 및 보완**

| 작업 내용 | 설명 |
| `사용상의주의사항`, `사용상주의사항`을 `사용시 주의사항`으로 통합 | 동일 성격의 정보가 컬럼마다 흩어져 있어 하나의 컬럼으로 통합 (변경된 컬럼 -> `사용시주의사항`) |
| 일부 약에만 존재하는 경우 `fillna()`로 보완 | 누락 정보 최대한 채움 |
| 통합 후 원본 열 삭제 | 중복 제거 및 단일화 |

### 4. **제품명 정제 및 용량 분리**
| 작업 내용 | 설명 |
| '제품명'에서 괄호, 특수문자, 중복 공백 제거 |
| 전처리된 결과는 '제품명_정제' 컬럼으로 분리 저장 → 검색어 입력/자동완성 기능에서 활용 가능 |


### 5. **성분정보 전처리 및 리스트화**
| '성분정보'에서 괄호, 숫자, 단위(mg 등)를 제거 |
| 줄바꿈, 슬래시, 쉼표 등으로 분리하여 성분명을 리스트화 → '성분명_리스트' 컬럼 생성|

### 6. **제품명 정제 및 용량 분리**
| 식약처 고시 및 실제 건강기능식품 성분을 바탕으로 영양성분 키워드 목록 정의 |
| '구분' == 일반의약품이고, '성분명_리스트'에 영양성분이 포함된 경우 → '영양제여부' = True |
| ➡ 사용자 복용 약품 중 영양제만 분리 가능하여 → 중복 섭취, 병용 가능성 판단 기능에 활용할 수 있음 |
