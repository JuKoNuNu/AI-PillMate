# 진행한 데이터 전처리
### 1. **필요 없는 행/열 제거 및 정리**

| 작업 내용 |
| `제품명`이 없는 행 제거 |
| `이미지_URL`, `이미지_설명`이 없는 행 제거 |
| `보험코드`, `분할선`, `사용상의주의사항(전문가)` 등 분석에 필요 없는 열 제거 |
| `제품명` 열 삭제 후 `이미지_설명`을 제품명으로 변경 |

### 2. **`크기` 열 분리**

| 작업 내용 | 설명 |
| `크기` → `장축`, `단축`, `두께`로 분리 | 모양 기반 식별에 유용 (예: 타원형, 두께 기준 검색 등) |

### 3. **주의사항 통합 및 보완**

| 작업 내용 | 설명 |
| `사용상의주의사항`, `사용상주의사항`을 `사용시 주의사항`으로 통합 | 동일 성격의 정보가 컬럼마다 흩어져 있어 하나의 컬럼으로 통합 (변경된 컬럼 -> `사용시주의사항`) |
| 일부 약에만 존재하는 경우 `fillna()`로 보완 | 누락 정보 최대한 채움 |
| 통합 후 원본 열 삭제 | 중복 제거 및 단일화 |

### 4. **제품명 정제 및 용량 분리**
| 작업 내용 | 설명 |
| `제품명`에서 용량 정보 (`50/850mg` 등) 추출 → `용량` 컬럼 생성 | 사용자 인식은 제품명 기준이므로 용량은 보조적 역할 |
| 용량 제거한 `제품명_정제` 컬럼 생성 | 예: `시타포민알파정` |
| 최종 표시용 `표시명` = 제품명_정제 + (용량) 생성 | 예: `시타포민알파정 (50/850mg)` → UI나 챗봇 응답에 활용 가능 |
